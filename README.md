# 태블로 시각화 스터디

 2024년 1학기에 진행한 **YBIGTA 시각화 스터디** 결과물을 업로드하는 repo입니다.
 - `교재`: **태블로 굿모닝 굿애프터눈**
 
 다음은 각 폴더에 대한 간단한 설명입니다.
 - `/n주차` : 각 주차별 공부 내용을 별도의 폴더로 구분
    - `n주차.md`: 내용 정리 및 결과물 관련 설명
    - `/result`: 결과 이미지 파일
    - `/images`: n주차.md 파일 작성을 위한 이미지
-  `/tableau_files`: 실습 태블로 파일을 주차별 폴더로 구분하여 저장
